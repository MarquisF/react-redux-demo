# react-redux-demo
a simple demo for test react-redux project

## Run the app
`npm run start`

## Run tests

run jest tests `npm test`
run jest with coverage `npm test -- --coverage`

## Test tools

> * [enzyme](http://airbnb.io/enzyme/index.html)
> * [jest](https://github.com/facebook/jest)
